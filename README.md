Multipost plug-in
===================

* Copyright:: Copyright 2012 Yuichi Takeuchi
* License:: GPL v2

MovableTypeであるブログに投稿した記事を同じ内容で、同一システム内の別のブログにマルチポストできるようにするプラグインです。

処理はシンプルで、投稿時に記事のクローンをつくり、別のブログの記事として登録します。記事アイテムも一緒にコピーされます。

MT 4.38でのみ動作確認。

なお、同梱のjQueryについてはそちらのライセンスに準じます。
